# Keyboard Shortcuts

## Global Shortcuts

### Opening the popup

- <kbd> Ctrl + Alt + T </kbd>

### Opening the sidebar

- <kbd> Ctrl + Alt + S </kbd>

## Popup Shortcuts

### Selecting a tab

- <kbd> Up </kbd>
- <kbd> Down </kbd>

### Selecting a tab group

- <kbd> Shift + Up </kbd>
- <kbd> Shift + Down </kbd>
