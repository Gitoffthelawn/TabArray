Class ContextualIdentity

The parameters for a contextual identity. This interface represents the containers defined in Firefox. For the containers displayed in the UI (may not be defined in the browser in rare cases), see ContainerAttributes.

See

https://developer.mozilla.org/en-US/docs/Mozilla/Add-ons/WebExtensions/API/contextualIdentities/ContextualIdentity
